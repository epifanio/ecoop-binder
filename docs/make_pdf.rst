    import os

-  Write Latex Document
~~~~~~~~~~~~~~~~~~~~~~~

.. code:: python

    texfile=''
    texfile += document
    texfile += abstract
    texfile += firstparagraph
    texfile += closedDocument
.. code:: python

    pdf = os.path.join(ID,'test.tex')
    f = open(pdf,'w')
    f.write(texfile)
    f.close()
-  Build PDF
~~~~~~~~~~~~

.. code:: python

    !pdflatex -output-directory={ID} {pdf}
